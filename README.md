# 📘 Resoluções dos Exercícios — *Introdução à Programação com Python*

Este repositório reúne as resoluções dos exercícios do livro  
**_Introdução à Programação com Python: Algoritmos e Lógica de Programação para Iniciantes_**,  
escrito por **Nilo Ney Coutinho Menezes**.

## 🧾 Sobre o livro

O livro é amplamente utilizado por iniciantes no estudo de programação, especialmente em cursos técnicos e de graduação no Brasil. Ele apresenta os fundamentos da lógica de programação com base na linguagem Python, abordando temas como:

- Variáveis e tipos de dados
- Operadores
- Estruturas de controle (condicionais e laços)
- Funções
- Listas e tuplas
- Manipulação de arquivos
- Orientação a objetos
- Bibliotecas padrão e módulos

Cada capítulo conta com explicações teóricas e diversos exercícios práticos, o que torna a obra ideal para quem deseja aprender de forma progressiva e estruturada.

## 🎯 Objetivo do repositório

O objetivo deste repositório é centralizar e organizar a resolução de **todos os exercícios propostos no livro**, capítulo por capítulo.  
Isso facilita a revisão, o acompanhamento do progresso e serve como apoio para outros estudantes que também estejam utilizando o livro.
